# 📷 AI Camera — умная камера с ИИ-фотографом

## Запуск

### 1. Установить зависимости
```bash
cd ai-camera
npm install
```

### 2. Запустить сервер
```bash
ANTHROPIC_API_KEY=sk-ant-... node server.js
```
На Windows (PowerShell):
```powershell
$env:ANTHROPIC_API_KEY="sk-ant-..."
node server.js
```

### 3. Открыть в браузере
- **Компьютер**: http://localhost:3000
- **Телефон**: открой `http://192.168.x.x:3000` (IP покажется в консоли при запуске)

> ⚠️ На iPhone камера работает только по HTTPS. На Android — по HTTP тоже работает.

### 4. Ввести API ключ
При первом открытии появится окно настроек — вставь свой Claude API ключ.
Получить ключ: https://console.anthropic.com

## Функции

| Функция | Описание |
|---|---|
| 📐 Сетка | Правило третей — кнопка ⊞ в камере |
| 🤖 AI Советник | Советы по кадру в реальном времени |
| 📸 Захват фото | Большая кнопка внизу |
| 🔄 AI Анализ | Автоматически после каждого снимка |
| 🎨 Фильтры | 7 фильтров: Тёплый, Холодный, Яркий, Ч/Б, Драма... |
| 🎛 Настройки | Яркость, контраст, насыщенность, теплота |
| 🗣 AI Чат | Скажи или напиши что хочешь изменить |
| 🎤 Голос | Нажми микрофон и говори по-русски |
| 💾 Сохранить | Скачать отредактированное фото |

## Модель

Используется `claude-haiku-4-5` — быстрая и дешёвая.
